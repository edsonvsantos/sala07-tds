Messi

Lionel Messi é um jogador argentino considerado um dos maiores da história do futebol. Conhecido por sua genialidade, dribles e precisão nos passes e finalizações, conquistou títulos com Barcelona, PSG, Inter Miami e a seleção argentina, incluindo a Copa do Mundo de 2022.

Tecnologias trabalhadas

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. É a base de quase todas as páginas que você vê na internet.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos HTML (ou outros documentos em XML).

JavaScript

JavaScript, frequentemente abreviado como JS, é uma linguagem de programação de alto nível, interpretada e baseada em objetos.

Uno Henrique

Uno Henrique é a versão automotiva e veloz de Bruno Henrique, atacante do Flamengo. No meme, ele vira um Uno tunado, simbolizando sua incrível velocidade em campo — “ó o homi passando de Uno!” virou sinônimo de arrancada imparável!
